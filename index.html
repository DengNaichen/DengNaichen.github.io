softmax

Posted on 2021-05-08
Symbols count in article: 482 Reading time ≈ 1 mins.

The softmax function is a function that turns a vector from real values to real values that sum to 1. The input can be positive, negitive, zero or greater than one. Softmax transfer them into values between 0 and 1, so that they can be interpreted as probabilities.

Softmax Formula

$$ \sigma(z){i} = \dfrac{e^{z{i}}}{\sum ^{K}{j=1}e^{z{j}}} $$

where $z_{i}$ are the elements of the input vactor and can take any real value. Denominator on the right hand side ensures that all the outputs values of the function will sum to 1, thus constituting a valid probability distribution.
