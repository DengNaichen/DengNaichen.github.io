Big Data and Data Processing

what data is:
we have tons of sensors to collect data. so we need a system or set of systems that helps us understanding this data and make sense of it.

Data -> Process -> Information

What is big data:

  • the volume of available data is huge
  • the data keeps growing at a staggering rate(a system digests high rate data)
  • data comes from variety of sources in totally different formats(integrating data)

The characteristics of scalable data processing

  • volume
  • velocity
  • variety

Basic Data Concept

Data Models

data model: collection of concepts for describing data
schema: description of a particular collection of data, using given data model.
Relational model of data:

  • most widely used model
  • relation is shown through the table with rows and columns.

Level of abstraction:
many views, single conceptual schema and physical schema.

  • views describe how users see the data
  • conceptual schema defines logical structure
  • physical schema describes the files and indexed used

One most important feature of DBMS: independence

  1. logical data independence: protects the user or the application on top of the database from changes in the logical schema or logical structure of the data.
  2. physical data independence: protect the application from changes in the physical schema or physical structure of the data.
